Overview: A Java-based application that manages a car rental business. It handles car inventory, customer data, rental transactions, and rental history using object-oriented programming principles.

Main Classes:

Main – Entry point; starts and runs the system.

RentalManagementSystem – Controls all operations (cars, customers, rentals).

Car – Stores car details (ID, model, year, rate, availability).

Customer – Holds customer info (ID, name, license, contact).

Rental – Connects a customer and a car; tracks dates and total cost.

++ 2 More added classes not included ++

Main Functionalities

Add and list available cars.
Register new customers.
Rent and return cars.
Display rental details and cost calculations.
Maintain and display system records.
The goal of this project is to simulate a real-world car rental management workflow using object-oriented programming principles.
